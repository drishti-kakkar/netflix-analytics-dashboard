# Netflix Streaming Intelligence Dashboard

A futuristic Netflix-inspired analytics dashboard built using Python, Streamlit, Plotly, and Pandas.

## Features

- Interactive Netflix analytics dashboard
- Dynamic filters
- KPI metrics
- Genre analysis
- Responsive UI
- Cinematic Netflix-inspired design

## Technologies Used

- Python
- Streamlit
- Pandas
- Plotly

## Run Locally

```bash
streamlit run app.py
```
